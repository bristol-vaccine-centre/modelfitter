# Set the font family and size in a huxtable globally

Set the font family and size in a huxtable globally

## Usage

``` r
.hux_set_font(hux, defaultFontSize = 8, defaultFont = "Roboto")
```

## Arguments

- hux:

  a huxtable table

- defaultFontSize:

  the desired font size

- defaultFont:

  the desired font

## Value

the altered huxtable
